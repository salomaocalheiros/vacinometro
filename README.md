![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square)

> Repositório do projeto **Vacinometro**.

## Tecnologias

- JavaScript

## Como rodar

### Pré-requisitos

- Node.js 18+ e npm

### Instalação

```bash
git clone https://github.com/salomaocalheiros/vacinometro.git
cd vacinometro
npm install
```

### Execução

```bash
npm start
```
